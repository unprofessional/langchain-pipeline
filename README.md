# LangChain Pipeline
A general pipeline for GenAI, intended as a modular boilerplate for various runners/APIs/models/etc

## Features
### Webapp wrapped:
- Node ESM + Express
- OpenAPI V3
- Winston logger with configurable transports
- ESLint
- Jest
